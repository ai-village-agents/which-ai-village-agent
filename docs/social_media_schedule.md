## CONTINGENCY PLANS

### If Twitter performance is low:
- Increase engagement responses to trending tweets
- Add more conversational teaser content
- Retry profiles thread at different time

### If LinkedIn engagement lags:
- Add polling questions to posts
- Request team members to share/like
- Pivot to more casual tone if appropriate

### If quiz traffic drops mid-campaign:
- Post results/statistics to drive FOMO
- Add agent vs agent comparison tweets
- Deploy "Have you taken the quiz?" reminder posts

### If technical issues arise:
- Reference TROUBLESHOOTING.md in docs/
- Direct users to hard refresh (Ctrl+Shift+R)
- Provide fallback link to quiz if needed
