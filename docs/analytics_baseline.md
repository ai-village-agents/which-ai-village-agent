## CONTINGENCY METRICS

### If Twitter Performance Lags
**Trigger:** <75 impressions in 6-hour window  
**Action:** 
- Increase engagement responses
- Post comparison tweets
- Tag @AIDigest for amplification
- **New Target:** 100 impressions by EOD

### If LinkedIn Underperforms
**Trigger:** <20 clicks from LinkedIn Day 4  
**Action:**
- Boost post with poll questions
- Request team member shares
- Adjust posting time to peak hours
- **New Target:** 30 clicks by EOD Day 5

### If Quiz Traffic Drops Mid-Campaign
**Trigger:** Day 5 sessions <100  
**Action:**
- Post result statistics to drive FOMO
- Deploy "Have you taken it yet?" reminder
- Share user testimonials
- **New Target:** 150+ sessions by EOD

### If Agent Distribution Skews
**Trigger:** Any agent <10 results by EOD Day 5  
**Action:**
- Review vector calibration (unlikely issueâ€”vectors locked)
- Check for regional preference biases
- Post spotlights for underperforming agents
- **Monitor:** Ensure all 11 agents represented

### If Negative Sentiment Emerges
**Trigger:** >20% negative comments  
**Action:**
- Address criticism with grace (see ENGAGEMENT_TEMPLATES.md)
- Clarify "fun quiz" vs "scientific" framing
- Escalate persistent trolling
- **Goal:** Maintain >80% positive sentiment
