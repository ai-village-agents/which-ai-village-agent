# AI Village Agent Personality Quiz - Visual Asset Catalog

## Overview

Complete suite of visual assets for the Day 300 personality quiz campaign. All assets are optimized for social media, web, and print usage.

---

## PERSONALITY CARDS (Individual Agents)

### Format
- **Dimensions**: 900 × 1200 pixels (3:4 ratio)
- **DPI**: 150 (high quality for printing)
- **Format**: PNG with transparent backgrounds (cards) / solid backgrounds (radars)
- **Naming**: `{agent_id}_card.png` and `{agent_id}_radar.png`

### Contents
Each personality card includes:
1. **Agent Name** (bold header)
2. **Archetype** (memorable descriptor)
3. **Tagline** (one-liner character description)
4. **Radar Chart** (6-dimensional personality visualization)
5. **Key Strengths** (5 bullet points)

## RADAR CHARTS (Standalone)

### Format
- **Dimensions**: 400 × 400 pixels (square)
- **DPI**: 150
- **Format**: PNG with transparent background
- **Use**: Flexible insertion into other designs

### 6 Dimensions Visualized
- **Structure** (0=Exploration → 1=Structure)
- **Verification** (0=Fast Iteration → 1=Strictness)
- **Abstraction** (0=Concrete → 1=Abstract)
- **Risk** (0=Conservative → 1=Risk-Taking)
- **Communication** (0=Dense/Technical → 1=Narrative/Accessible)
- **Collaboration** (0=Pairing/Iterative → 1=Broadcast)

### Use Cases
- **Profile Images**: Agent avatars on Twitter/LinkedIn
- **Comparison Posts**: Side-by-side agent profile posts
- **Blog Headers**: Substack article hero images
- **Merchandise**: T-shirts, mugs, posters

## TECHNICAL SPECIFICATIONS

### File Format
- **Format**: PNG (lossless)
- **Color Space**: RGB
- **Transparency**: Yes (for radar/card overlays)
- **Compression**: Optimized for web

### Size Optimization
All assets are optimized for web:
- Individual cards: 150-300 KB
- Radar charts: 50-100 KB
- Comparative visualizations: 200-400 KB
- Total asset size: ~3 MB (all files)

### Regeneration
All assets auto-generate from source data:
```bash
python3 generate_cards.py                # Individual cards + radars
python3 generate_dimension_grid.py       # Dimension pair grids
python3 generate_all_agents_radar.py     # Combined overlay radar
python3 generate_marketing_assets.py     # Teaser, matrix, comparison
```
